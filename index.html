ClimbTracker

ClimbTracker 是一款专为攀岩爱好者设计的 Android 应用，旨在帮助用户轻松记录、跟踪和分析他们的攀岩活动。

无论您是抱石还是绳索攀岩，此应用都能让您详细记录每一次的攀爬，并通过直观的日历和图表来回顾您的进度。

✨ 核心功能

📅 日历视图

  • 训练热图: 每日的攀岩地点会直接显示在日历上，攀爬数量越多的日期热图面积越大。
  • 日历跳转: 对于有攀爬记录的日期，点击热图可以跳转至当日的攀爬记录页面。
  • 添加 Session: 您可以直接在日历上为选定的日期创建新的攀岩 Session（活动）。

🧗‍♀️ 记录列表

  • 详细记录: 按日期和 Session 分组，清晰地展示您的每一次攀爬尝试。
  • 完整操作: 您可以轻松添加、编辑或删除任何一条攀爬记录。
  • 排序功能: 在每个 Session 内部，您可以使用右上角的排序按钮，按难度或完成状态对记录进行排序。

📊 统计分析

  • 多维度筛选: 按时间范围（本周、本月、全部）和攀岩类型（抱石、绳索）筛选数据。
  • 完攀统计: 以图表形式展示您在不同难度等级下的完攀数量和平均尝试次数。
  • 趋势分析: 通过折线图查看您的训练负荷和攀爬效率在最近几次活动中的变化趋势。

📖 使用说明

1. 下载与安装

点击下方的“下载安卓应用”按钮，下载 APK 文件。下载完成后，在您的安卓设备上打开文件并根据提示完成安装。

2. 创建攀岩活动 (Session)

在“日历”界面，选择您进行攀岩的日期，然后点击“添加 Session”按钮。输入攀岩地点、日期等基本信息，即可创建一个新的活动记录。

3. 添加攀爬记录

进入“记录”界面，您可以添加每一次的攀爬，详细记录难度、完成状态以及尝试次数。数据会被自动分类到日期最新的session中。短按每条攀爬记录可以进行修改，长按进行删除。

4. 查看统计数据

在“统计”界面，您可以筛选并查看不同时间范围和攀岩类型的统计图表，帮助您分析自己的表现和进步。
